How should users switch between devices?

The Radar screen currently shows only the first device. Users need a way to cycle through all their deployed sensors (e.g., "North Gate Sensor", "South Boundary", "East Trail Monitor").

A) Swipe Left/Right

Horizontal swipe gesture on the map area, like swiping between pages. Page dots show position. Feels native on mobile — same gesture as Photos or onboarding screens.

Pros

  • Most natural mobile gesture
  • Full screen stays clean
  • Discoverable via page dots

Cons

  • Conflicts with map pan gesture
  • Can't jump to device #4 directly
  • No device names visible until you land

B) Dropdown Picker

Tap the device name in the status bar to open a dropdown. Shows all devices with online/offline status. Tap to select. Direct jump to any device.

Pros

  • See all devices + status at once
  • Jump directly to any device
  • No gesture conflicts with map

Cons

  • Extra tap to open + tap to select
  • Dropdown overlays the map
  • Less "tactile" than swiping

C) Scrollable Chip Bar ⭐ Recommended

Horizontal row of device chips below the Live Map / Replay toggle. Each chip shows device name + status dot. Tap to switch. Always visible, no extra taps needed.

Pros

  • All devices visible at a glance
  • Single tap to switch
  • Online/offline status always visible
  • No gesture conflicts with map

Cons

  • Uses ~40px vertical space
  • Chips truncate with many devices
