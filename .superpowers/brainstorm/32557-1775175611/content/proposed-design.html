Changes Summary

Fix 1 — Shrink branding: Remove 80px logo and "TrailSense AI" title (already in header). Keep only the one-line subtitle as context. Reduce brandingContainer from paddingVertical: 32 to paddingVertical: 16, remove paddingTop on subtitle. Saves ~150px of vertical space.
Fix 2 — Keyboard: The iOS simulator disables the software keyboard by default (toggle with I/O > Keyboard > Toggle Software Keyboard or Cmd+K). On real devices the TextInput already works. No code change needed — document the simulator behavior.
Fix 3 — Scrollable welcome: Wrap the welcome content (SecurityStatusCard + branding + SuggestionChips) in a ScrollView so all suggestions are reachable. Add bottom padding to the scroll content to clear the input bar. The SafeAreaView edges stays ['top'] only — the tab bar already handles bottom insets.
